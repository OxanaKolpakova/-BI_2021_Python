Написать утилиту для работы с последовательностями нуклеиновых кислот.


Необходимо реализовать программу, которая в бесконечном цикле считывает команды от
пользователя. После команды программа должна запрашивать у пользователя
последовательность нуклеиновой кислоты и распечатывать результат.
Список команд:
exit — завершение исполнения программы
transcribe — напечатать транскрибированную последовательность
reverse — напечатать перевёрнутую последовательность
complement — напечатать комплементарную последовательность
reverse complement — напечатать обратную комплементарную последовательность
+ любые дополнительные команды на ваш страх и риск
Требования:
Программа должна сохранять регистр символов (напр. complement AtGc это TaCg)
Программа должна работать только с последовательностями нуклеиновых кислот. К примеру, последовательность AUTGC не может существовать, так как содержит T и U, такие случаи нужно обрабатывать и сообщать об этом пользователю (см. пример).
Запрещается использование сторонних модулей
Правильно оформить гитхаб.
Пример работы программы в прикреплённой картинке.

Как сдать задание?
Создайте ветку в которой вы будете выполнять ДЗ (название любое). Создавайте ветки и делайте коммиты у себя в терминале.
Запушьте ваше решение в виде python скрипта на гитхаб (добавление README приветствуется).
Создайте пулл-реквест из вашей ветки с ДЗ в ветку main. Про pull requestы можно почитать здесь (https://guides.github.com/activities/hello-world/), начиная с 4 степа, там нет ничего сложного.
Отправьте ссылку на пулл-реквест в ответ на это задание в гугл-классах. Ссылка будет иметь следующий вид: https://github.com/profile_name/repository_name/pull/1. Обратите внимание, что ваш репозиторий должен быть публичным! Комментарии относительно вашего кода будут оставлены прямо в пулл-реквесте, по этому не нужно закрывать пулл-реквест (merge pull-request) до проверки преподавателем.
